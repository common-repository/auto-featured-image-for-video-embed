=== Video Thumbnail Auto-Feature ===
Contributors: jdingman 
Tags: featured image, thumbnails, video, embed
Requires at least: 3.3.2
Tested up to: 3.4.1
Stable tag: 1.5

This plugin was built to download and set the featured image of a thumbnail for specific video source embeds.  Currently supported video sources:  YouTube, WordPress.tv


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
